This tool uses data from Sentinel-1 C-band SAR and Landsat 8 to provide accurate insights into land cover. Here‚Äôs a brief overview of these data sources:

- **Sentinel-1:** A European space mission that provides high-resolution radar images. It can penetrate clouds and works day and night, making it an excellent choice for land monitoring.
- **Landsat 8:** An American satellite that captures both optical and infrared images of Earth's surface. It allows for detailed analysis of vegetation, water, and land use patterns.
